# Removedor de Fundo com Streamlit

Este projeto utiliza **Streamlit** e a biblioteca **rembg** para remover o fundo de imagens de forma simples e rápida. A aplicação roda inteiramente em Python, sem necessidade de um backend separado.

---

## Tecnologias utilizadas
- [Streamlit](https://streamlit.io/) — Interface web interativa
- [rembg](https://github.com/danielgatis/rembg) — Biblioteca de IA para remoção de fundo
- [Pillow (PIL)](https://python-pillow.org/) — Manipulação de imagens
